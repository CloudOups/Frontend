Ajouter un événement

Le nom de l'événement est requis.
Le nom de l'événement doit comporter au moins 6 caractères.
La catégorie est requise.
La date de début est requise et doit être au format (AAAA-MM-JJTHH:MM).
La date de fin est requise et doit être au format (AAAA-MM-JJTHH:MM).
Le nombre de participants est requis.
Le nombre de participants doit être d'au moins 2.
Le nombre de participants doit être un nombre valide.
Le lieu est requis.
Image requise
